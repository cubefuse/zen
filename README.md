# Cubefuse Zen

> The Open-Source 2-Factor Authentication (2FA) App

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development in browser
```
yarn run serve
```

### Compiles, minifies and copies bundle to platforms
```
yarn run build
```

### Opens the app in Platform IDEs
```
yarn run open
```

### Lints and fixes files
```
yarn run lint
```
